# forget gate

h t-1과 xt를 보고 memory cell의 장기기억에서 지워도 될 내용을 지움.

# input gate

1. i필터: h t-1과 xt를 보고 이번 새로운 내용을 추가할지 결정.
2. C필터: tanh (-1 ~ 1)로 이 정보가 어떤 내용인지 결정.
