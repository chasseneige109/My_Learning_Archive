# **Scheduled Sampling**

교수님이 자전거 비유로 설명한 부분의 핵심이 이거임.

### ❗ Scheduled Sampling 아이디어

- 처음엔 **Teacher Forcing 100%**
    
- 조금 학습되면 **정답 단어 90%, 모델 출력 10%**
    
- 더 학습되면 **정답 50%, 모델 출력 50%**
    
- 결국 **정답 없이 모델 스스로 생성하도록 학습됨**
    

이 비율이 바로:

> **Teacher Forcing Ratio**

비율을 점점 줄여서  
“아빠가 점점 손을 놓는 것처럼” 만드는 것.
