명확한 용어 정리가 필요하시군요. '보관함' 같은 비유를 배제하고, 파이토치(PyTorch) 소스 코드 레벨에서의 **정식 명칭과 클래스 간의 계층 구조**를 정확하게 기술해 드리겠습니다.

---

### 1. 객체 모델 계층 구조 (Object Hierarchy)

파이토치 모델은 **컴포지션(Composition)** 구조로 이루어져 있으며, 각 단계의 정식 명칭은 다음과 같습니다.

1. **Top-level Module (Custom Class)**: 사용자가 정의한 `Net` 인스턴스입니다. `nn.Module`을 상속받습니다.
    
2. **Sub-Module (Member Attribute)**: `Net` 내부의 `self.fc1`에 할당된 `nn.Linear` 인스턴스입니다. 이 역시 `nn.Module`을 상속받은 객체입니다.
    
3. **Parameter (Leaf Node)**: `nn.Linear` 내부에 존재하는 `self.weight`와 `self.bias`입니다. 이들의 클래스는 `torch.nn.parameter.Parameter`입니다.
    

---

### 2. 내부 데이터 구조와 등록 메커니즘

각 인스턴스는 부모인 `nn.Module`로부터 물려받은 **내부 속성(Internal Attributes)**을 통해 하위 요소를 관리합니다.

#### **(1) `Net` 인스턴스 내부**

- **속성명**: `self._modules`
    
- **타입**: `collections.OrderedDict[str, Optional[Module]]`
    
- **설명**: `self.fc1 = nn.Linear(...)`를 실행하면, `nn.Module.__setattr__` 메서드가 작동하여 해당 `Linear` 인스턴스를 `self._modules['fc1']`에 등록합니다.
    

#### **(2) `nn.Linear` 인스턴스 내부**

- **속성명**: `self._parameters`
    
- **타입**: `collections.OrderedDict[str, Optional[Parameter]]`
    
- **설명**: `nn.Linear`의 클래스 정의 내부(`__init__`)에서 `self.bias = Parameter(...)`가 실행될 때, 이 값은 `self._parameters['bias']`에 등록됩니다.
    

---

### 3. 경로 탐색의 정식 프로세스 (Full Path)

사용자가 `model.fc1.bias`라고 호출할 때의 정식 탐색 순서는 다음과 같습니다.

1. **`model.__getattr__('fc1')`** 호출
    
    - `model._modules`에서 `'fc1'` 키를 검색하여 `nn.Linear` 인스턴스를 반환합니다.
        
2. **`linear_instance.__getattr__('bias')`** 호출
    
    - `linear_instance._parameters`에서 `'bias'` 키를 검색하여 `nn.parameter.Parameter` 인스턴스를 반환합니다.
        
3. **`Parameter` 객체 접근**
    
    - 최종적으로 `torch.Tensor`를 상속받은 `Parameter` 객체에 도달하여 데이터 값과 기울기(`grad`)를 참조합니다.
        

---

### 4. 핵심 요약 표

|**계층 (Level)**|**정식 클래스 (Class)**|**소속된 내부 속성 (Attribute)**|**데이터 타입 (Type)**|
|---|---|---|---|
|**Root**|`__main__.Net`|`self._modules`|`OrderedDict`|
|**Child**|`torch.nn.modules.linear.Linear`|`self._parameters`|`OrderedDict`|
|**Leaf**|`torch.nn.parameter.Parameter`|`self.data`, `self.grad`|`torch.Tensor`|

---

### 결론

정리하자면, **`Net` 인스턴스의 `_modules` 속성(OrderedDict) 안에 `nn.Linear` 인스턴스가 등록되어 있고, 그 `nn.Linear` 인스턴스의 `_parameters` 속성(OrderedDict) 안에 `bias`라는 `Parameter` 인스턴스가 들어있는 구조**입니다.
